# HB-Study

HB-Study is a link list page which was developed by React.js and React-Bootstrap.

## Clone

HB Study

```
https://github.com/berksafran/hb-study.git
```

## Install

```
cd hb-study
yarn install
or
npm install
```

## Usage

1- Launch HB-Study
```
cd hb-study
yarn start
```
## Test

1-  Jest & Enzyme Unit Test

```
cd hb-study
yarn test
```

2- E2E Cypress Test

```
yarn cypress run
```
